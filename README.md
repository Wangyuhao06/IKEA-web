> **Silent Leaks: Implicit Knowledge Extraction Attack on RAG Systems Through Benign Queries**

A stealthy attack framework that extracts RAG knowledge using natural-looking queries, bypassing input/output-level defenses that block traditional prompt injection and jailbreak attacks.

## 🛠️ Deploy to GitHub Pages

### Step 1: Create Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it `IKEA-poster` (or any name you prefer)
3. Make it **Public**
4. Do NOT initialize with README (we already have one)

### Step 2: Upload Files

#### Option A: Web Upload (Easiest)
1. Open your new repository on GitHub
2. Click **"Add file"** → **"Upload files"**
3. Drag and drop all 5 files:
   - `index.html`
   - `image.png`
   - `image(1).png`
   - `image(2).png`
   - `qr_code.png`
4. Click **"Commit changes"**

#### Option B: Git Command Line
```bash
# Clone your repository
git clone https://github.com/YOUR_USERNAME/IKEA-poster.git
cd IKEA-poster

# Copy all files to this folder
# Then commit and push
git add .
git commit -m "Initial commit: IKEA ICLR 2026 poster"
git push origin main
```

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **"Settings"** tab
3. Scroll down to **"Pages"** section (left sidebar)
4. Under **"Source"**, select:
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
5. Click **"Save"**

### Step 4: Access Your Site

- Wait 1-2 minutes for deployment
- Your site will be at: `https://YOUR_USERNAME.github.io/IKEA-poster`
- You can find the exact URL in the Settings → Pages section
